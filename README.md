# Gistub

Gistub is a stand alone application for sharing snippet such as `gist.github.com`.

## How to use

```sh
git clone git://github.com/seratch/gistub.git
cd gistub
bundle install
rake db:migrate
rails s
```

Access `http://localhost:3000/` through web browser.
